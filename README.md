# VUE3 练手

Vue 3 + Vite 项目，包含 Modal 弹窗组件演示。

## 技术栈

- Vue 3 + TypeScript
- Vite 7
- Pinia + Vue Router
- commitlint + husky（Conventional Commits）

## 项目启动

```sh
npm install
npm run dev
```

## 构建

```sh
npm run build
```

## 代码检查

```sh
npm run lint
```

## Git 提交规范

使用 Conventional Commits 规范：

```sh
npm run commit   # 交互式提交
# 或直接写格式：<type>(<scope>): <description>
```
